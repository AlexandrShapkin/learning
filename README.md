# Learning

Личный репозиторий практических учебных интенсивов по структуре базы знаний [knowledge](https://github.com/AlexandrShapkin/knowledge).

`knowledge` хранит обобщённый результат обучения. Этот репозиторий хранит подготовленные маршруты, задания, лабораторные работы и практические артефакты.

## Структура

```text
learning/
├── README.md
├── RULES.md
├── AGENTS.md
├── docs/
├── templates/
└── sections/
    ├── 01-Вычислительные системы/
    ├── 02-Операционные системы/
    ├── 03-Сети и протоколы/
    ├── 04-Программирование и среды выполнения/
    ├── 05-Алгоритмы и структуры данных/
    ├── 06-Данные и хранилища/
    ├── 07-Прикладные системы и интерфейсы/
    ├── 08-Проектирование программного обеспечения/
    └── ...
```

Наполненный раздел содержит `README.md` и каталог `intensives/`. Основной интенсив содержит общий маршрут, ресурсы, описание окружения, scripts и последовательность спринтов. Тематические углубления добавляются как отдельные интенсивы.

## Как проходить

1. Выбрать раздел и интенсив.
2. Последовательно проходить спринты.
3. Изучать подготовленный основной маршрут ресурсов.
4. Выполнять задания и лабораторные работы.
5. Сохранять полезные практические результаты.
6. Переносить обобщённые знания в `knowledge`.

Пользователю не требуется самостоятельно придумывать задания, искать основные источники или вести формальные таблицы прогресса.
