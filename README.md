# Learning

Личный репозиторий практических учебных интенсивов по структуре базы знаний [knowledge](https://github.com/AlexandrShapkin/knowledge).

База знаний хранит обобщённый и пригодный для повторного использования результат обучения. Этот репозиторий хранит готовые маршруты, задания, лабораторные работы и практические артефакты.

## Устройство

```text
learning/
├── README.md
├── RULES.md
├── AGENTS.md
├── docs/
│   └── CONCEPT.md
├── templates/
│   ├── README.md
│   ├── intensive.md
│   └── sprint.md
└── sections/
    ├── README.md
    ├── 01-Вычислительные системы/
    ├── 02-Операционные системы/
    ├── 03-Сети и протоколы/
    └── ...
```

Структура наполненного раздела:

```text
<Раздел>/
├── README.md
└── intensives/
    ├── 01-Основной интенсив/
    │   ├── README.md
    │   ├── RESOURCES.md
    │   ├── LAB_ENVIRONMENT.md  # когда требуется отдельная лаборатория
    │   ├── scripts/
    │   └── sprints/
    └── 02-Тематическое углубление/
```

Пустые каталоги `intensives/` заранее не создаются. Они появляются вместе с первым полноценным интенсивом.

## Как проходить

1. Выбрать раздел и интенсив.
2. Последовательно проходить спринты.
3. Изучать подготовленный основной маршрут ресурсов.
4. Выполнять задания и лабораторные работы.
5. Сохранять только полезные практические результаты.
6. Переносить обобщённые знания в репозиторий `knowledge`.

Пользователю не требуется придумывать задания, искать основные источники или вести формальные таблицы прогресса.

## Доступные интенсивы

- [Основы вычислительных систем](sections/01-Вычислительные%20системы/intensives/01-Основы%20вычислительных%20систем/) — представление данных, процессор, память, выполнение программ, ввод-вывод, конкурентность, производительность и виртуализация.
- [Основы операционных систем и Linux](sections/02-Операционные%20системы/intensives/01-Основы%20операционных%20систем%20и%20Linux/) — ядро и user space, загрузка, процессы, память, файлы, storage, users, packages, systemd, network, isolation, security, diagnostics и recovery.
- [Основы сетей и протоколов](sections/03-Сети%20и%20протоколы/intensives/01-Основы%20сетей%20и%20протоколов/) — packet flow, Ethernet, IPv4/IPv6, routing, UDP/TCP, DNS, DHCP, HTTP, TLS, proxy, load balancing, VPN, overlay networks и диагностика.

## Разделы

Список областей находится в [`sections/README.md`](sections/README.md). Названия и порядок разделов соответствуют верхнему уровню базы знаний.

## Документы

- [`RULES.md`](RULES.md) — обязательные правила реализации и прохождения.
- [`docs/CONCEPT.md`](docs/CONCEPT.md) — полная концепция репозитория.
- [`AGENTS.md`](AGENTS.md) — инструкции для ИИ-агентов, создающих интенсивы.
- [`templates/`](templates/) — шаблоны будущих интенсивов и спринтов.
