# Прикладные системы и интерфейсы

Практическое изучение браузеров, frontend, backend, API, аутентификации, авторизации, интеграций и фоновых процессов.

Раздел соответствует области [«Прикладные системы и интерфейсы»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Прикладные%20системы%20и%20интерфейсы) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Практика должна приводить к работающим приложениям и проверяемым пользовательским сценариям.
