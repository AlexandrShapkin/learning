# Программирование и среды выполнения

Практическое изучение языков, систем типов, памяти, ошибок, конкурентности, компиляции, интерпретации, runtime и инструментов разработки.

Раздел соответствует области [«Программирование и среды выполнения»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Программирование%20и%20среды%20выполнения) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Go используется как основной язык, а другие языки — для сравнения моделей и изучения специфических механизмов.
