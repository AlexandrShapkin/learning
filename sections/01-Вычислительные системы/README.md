# Вычислительные системы

Практическое изучение представления данных, процессоров, памяти, выполнения программ, ввода-вывода, конкурентности и аппаратной виртуализации.

Раздел соответствует области [«Вычислительные системы»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Вычислительные%20системы) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Основной интенсив должен сформировать целостную модель того, как компьютер физически и логически выполняет программы.
