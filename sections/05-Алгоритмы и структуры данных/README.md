# Алгоритмы и структуры данных

Практическое изучение вычислительной сложности, структур данных, поиска, сортировки, деревьев, графов и алгоритмических подходов.

Раздел соответствует области [«Алгоритмы и структуры данных»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Алгоритмы%20и%20структуры%20данных) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Практика должна включать собственные реализации, измерения и разбор выбора алгоритма, а не только решение задач на платформах.
