# Наблюдаемость и диагностика

Практическое изучение логов, метрик, трассировок, профилирования, health checks, оповещений, производительности и поиска первопричин.

Раздел соответствует области [«Наблюдаемость и диагностика»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Наблюдаемость%20и%20диагностика) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Центральной формой практики являются подготовленные неисправности, сбор сигналов и доказательная диагностика.
