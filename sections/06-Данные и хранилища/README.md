# Данные и хранилища

Практическое изучение моделирования данных, SQL, транзакций, индексов, репликации, кешей, поиска и резервного восстановления.

Раздел соответствует области [«Данные и хранилища»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Данные%20и%20хранилища) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Основной маршрут должен связывать модель данных с фактическим поведением хранилища и планами выполнения.
