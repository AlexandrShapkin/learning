# Безопасность

Практическое изучение угроз, идентификации, доступа, криптографии, защиты сетей, ОС, приложений, контейнеров и облачной инфраструктуры.

Раздел соответствует области [«Безопасность»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Безопасность) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Все задания выполняются только в контролируемом окружении и должны сочетать предотвращение, обнаружение и разбор последствий.
