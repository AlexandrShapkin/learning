# Инфраструктура и платформы

Практическое изучение физической и виртуальной инфраструктуры, контейнеров, Docker, облаков, Infrastructure as Code, Terraform и Kubernetes.

Раздел соответствует области [«Инфраструктура и платформы»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Инфраструктура%20и%20платформы) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Окружение должно быть воспроизводимым, а задания — включать отказные сценарии и очистку ресурсов.
