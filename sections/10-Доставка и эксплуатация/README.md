# Доставка и эксплуатация

Практическое изучение сборки, тестирования, артефактов, CI/CD, релизов, развёртывания, конфигурации, секретов и восстановления.

Раздел соответствует области [«Доставка и эксплуатация»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Доставка%20и%20эксплуатация) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Практика должна охватывать весь путь изменения от исходного кода до работающей и сопровождаемой системы.
