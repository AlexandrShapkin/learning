# Инженерный процесс

Практическое изучение Git, тестирования, code review, документации, зависимостей, автоматизации и принятия технических решений.

Раздел соответствует области [«Инженерный процесс»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Инженерный%20процесс) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Практика должна улучшать реальный процесс изменения программного продукта, а не имитировать формальные церемонии.
