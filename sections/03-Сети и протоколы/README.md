# Сети и протоколы

Практическое изучение передачи данных, Ethernet, IP, маршрутизации, TCP, UDP, DNS, TLS, HTTP, балансировки и сетевой диагностики.

Раздел соответствует области [«Сети и протоколы»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Сети%20и%20протоколы) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Основной маршрут должен сопровождаться захватом трафика, построением лабораторных сетей и диагностикой неисправностей.
