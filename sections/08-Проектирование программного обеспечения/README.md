# Проектирование программного обеспечения

Практическое изучение модульности, зависимостей, качества кода, шаблонов, архитектурных стилей, предметного моделирования и решений.

Раздел соответствует области [«Проектирование программного обеспечения»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Проектирование%20программного%20обеспечения) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Задания должны требовать сравнения вариантов, эволюции дизайна и фиксации обоснованных решений.
