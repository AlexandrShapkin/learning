# Операционные системы

Практическое изучение управления ресурсами, процессов, памяти, файловых систем, пользователей, прав, служб и системного администрирования.

Раздел соответствует области [«Операционные системы»](https://github.com/AlexandrShapkin/knowledge/tree/main/content/Операционные%20системы) базы знаний.

Будущие интенсивы будут размещаться в `intensives/`. Основной интенсив должен опираться на Linux и раскрывать механизмы ОС через наблюдение и диагностику.
