# Спринт 09. Value proposition, vision и positioning

Ориентировочный объём: 18–27 часов.

## Центральный вопрос

Для кого, в каком контексте и почему продукт должен быть предпочтительнее текущих альтернатив?

## Результат спринта

После завершения необходимо уметь:

- формулировать value proposition by segment and job;
- distinguish product vision, strategy, positioning and messaging;
- identify alternatives including manual work and non-consumption;
- describe differentiated benefit and evidence;
- avoid feature lists as value proposition;
- state target customer and excluded contexts;
- connect user outcome to organizational value;
- test positioning with comprehension and choice evidence;
- separate aspiration from current capability.

## Основной маршрут

1. Select one validated segment/job.
2. Map alternatives, pains, desired outcomes and differentiated value.
3. Write vision and positioning statements.
4. Test clarity against synthetic buyer/user responses.

## Лабораторная работа

Создать `work/value-positioning/`.

### Requirements

- target segment and buying/use context are explicit;
- current alternatives and switching costs are represented;
- value statement names outcome rather than feature;
- differentiated claim has evidence or validation plan;
- buyer and user value are both covered;
- vision is stable enough while strategy remains adaptable;
- non-target contexts prevent overgeneralization;
- messaging does not promise unsupported capability.

### Failure scenarios

1. Value proposition is a feature inventory.
2. Target audience is “everyone”.
3. Alternative is assumed to be only a direct competitor.
4. Buyer value conflicts with daily user experience.
5. Positioning claim has no evidence.
6. Vision and roadmap are confused.

## Результат для базы знаний

- **Product value propositions**.
- **Product vision, strategy and positioning**.
- **Alternatives, switching costs and differentiated value**.
- **Buyer and user value alignment**.

## Когда переходить дальше

Можно переходить, когда a target segment can understand the promised outcome, compare it with real alternatives and recognize why the product is relevant.
