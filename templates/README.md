# Шаблоны

Каталог содержит структурные шаблоны для разработки будущих материалов.

- [`intensive.md`](intensive.md) — паспорт и маршрут интенсива.
- [`sprint.md`](sprint.md) — полностью подготовленная инструкция спринта.

Шаблоны используются авторами и ИИ-агентами. Пользователь не должен заполнять их во время прохождения.
